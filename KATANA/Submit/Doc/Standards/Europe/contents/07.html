7. Demos

In the following guidelines, a Story Demo is an explanation of the game story mainly using movies, etc., and a Play Demo is an automatic demonstration of actual game play (the main game).

7.1 Screen Display

Required: During the Play Demo sequence, "PRESS START BUTTON," should be displayed on the screen to clearly distinguish from the main game (after game start).

Note: Story demos are not reflected in the above-mentioned required item.

7.2 Advancing to the Next Screen

Required: Pressing the START button during either the Play Demo or Story Demo should immediately bring up the Title Screen.

If the START button is not pressed, end the demo after a short time has elapsed, and proceed to the next screen of the title loop.

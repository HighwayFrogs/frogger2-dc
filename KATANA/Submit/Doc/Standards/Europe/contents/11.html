Dreamcast Software Development Standards/Ver. 2.00EU

11. Score/High Score/No. of Players

11.1 Score / High Score Indication

Reference Example: Score / High Score Indication

"SCORE"

"PLAYER 1/PLAYER 2" or "1P/2P"

"HI SCORE", "HIGH SCORE" or "TOP"

11.2 No. of Players Indication

Recommended: The machine currently being operated by the player should not be counted in an indication of the number of remaining players.

When the number of remaining machines is indicated as 2, we recognize that there are two remaining machines in addition to "the machine currently being operated by the player." So if one machine loses and the game is over, the number of remaining machines is indicated as 0.

11.3 High Score Retention

Recommended: Be sure that the high score is not initialized by a soft reset during main game play.

This applies except for the case of returning to the Main Unit Boot ROM Menu, such as a soft reset from the Title Loop.

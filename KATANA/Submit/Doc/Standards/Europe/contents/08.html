8. Start/Option Selection

8.1 Difference from the Title Screen (Game Start Screen)

Required: The Start/Option Selection screen exists separately from the Title Screen.

If a "Menu" selection such as Start/Option is needed, it should appear when the START button is pressed at the Title Screen. Objects other than the Menu can be used and combined as needed.

8.2 Start Menu Selection Limitations

Required: According to the controller connection state, items that are unusable should be displayed in the menu, but not be selectable.

In a 2-player fighting game with a controller connected only to port A, menu fighting mode selections such as "vs." should not be selectable. In this case, to provide visual indication of the unselectable state: if the menu selection is normally displayed as white it would appear grey, and would be inaccessible to the cursor.

Determining the connection status of the controller is important here. When a telephone line is not connected, not being able to proceed to "Network Mode" is not required.

8.3 Advancing to the Next Screen

Recommended: Pressing the START button should advance to the main game or option selected in each menu.

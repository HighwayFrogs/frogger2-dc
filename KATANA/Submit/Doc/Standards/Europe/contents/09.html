Dreamcast Software Development Standards/Ver. 2.00EU

9. Options

Recommended: Options should be provided whenever possible, unless they would destroy the character of the game or if the game is a transplant of an original that has no options.

Reference Example: In race or action games where it would disturb the user to have the Start/Option button bring up the Options Screen and then return to the Title, such as in an RPG/Adventure, the options can be set by an Options command from within the game.

9.1 Options

Reference Example: The following options are standard for arcade transplants and others.

LEVEL (of difficulty)        EASY >NORMAL< HARD

PLAYERS (number of)          1 2 >3< 4 5

CONTROL (button setting)

RAPID (continuous shooting)  >ON< OFF

AUDIO (output setting)       >STEREO< MONAURAL

SOUND (test)                 >BGM01<

                             SE02

                             VOICE00

Initialise Settings (return to initial settings)

EXIT

Prohibited Terms: "Default" or "Reset" to refer to the initialisation of settings and the initial settings.

Required: Considering the many users who play on monaural TVs, the Stereo/Monaural selection should be provided whenever possible.

9.2 Storing Option Settings

Recommended: Option settings should be backed up (saved) if possible.

Required: If the option settings are to be backed up according to the above, the Stereo/Monaural setting should not be saved (masked from the save data).

In Dreamcast, the Stereo/Monaural settings are saved in the Main Unit, considering the case where the same data is to be used to play in another environment.

Recommended: Despite the existence of the backup data, the option settings should be retained after soft reset.

9.3 Stereo/Monaural Audio Output Settings

The initial Stereo/Monaural setting should match that in the Boot ROM as mentioned in 3.1 Audio Output Settings Upon Loading an Application.

Required: When the user changes the value of the voice output setting on the application option screen, the Main Unit Boot ROM setting should also be set to the same value before the option screen is exited.

Change the Boot ROM setting when exiting from the option screen. Changing the Boot ROM setting each time, adjusting to the cursor movement of the user when he or she is choosing the stereo/monaural setting, should not be done.

As used in the procedures of 3.1, 9.2 and 9.3, the Stereo/Monaural setting refers to Main Unit setting, and other option settings refer to the saved data (if present).

9.4 50/60Hz Option

 Required: If a 50/60 Hz option is included in the game options screen, refer to section 29.

9.5 Proceeding to the next screen

Recommended: When EXIT is selected and any action button is pressed, or when a Start/Option button is pressed from any location in the Options Screen after a Start/Option setting, operation should revert to the Start/Option Selection screen instead of to the Sega Logo and Title Screen.

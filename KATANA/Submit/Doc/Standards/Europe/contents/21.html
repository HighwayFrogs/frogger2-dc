20 Rules for Dreamcast-Specific Disc Single/High Density Area

On the Dreamcast-specific disc, the part that can be read by the CD player is referred to as the "Single Density Area" and the part that cannot been read is referred to as the "High Density Area".

20.1 DA Regulations in the Single Density Area

Required: In the single density area of the Dreamcast-specific disc, DA except for non-sound should be input as voice data. DA sound should be made to run even if the specific-disc is put in a non-Dreamcast CD player.

Recommended: Voice data input in the single density area can be created by each application, but when voice data is not originally prepared, the sample in this company's SDK should be used.

Take care to avoid prohibited terms when creating a warning announcement such as "This is a Dreamcast-specific disc. Data for special use with Dreamcast is entered on this disc, so do not play it on a standard CD player".

Prohibited Terms: "GD", "GD-ROM" (these should just be called "discs/disks")

20.2 Prohibitions of the Writing Area

Required: The Dreamcast program execution code cannot appear in an area other than the Dreamcast-specific disc high-density area.

For security reasons, the Dreamcast program execution code cannot be written to any area other than the Dreamcast-specific disc double-density area. Dreamcast program execution code should not be written to Dreamcast-specific disc single-density area, save data, or Visual Memory special games.

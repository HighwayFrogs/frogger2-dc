Dreamcast Software Development Standards/Ver. 2.00E

28. Applications for Operation on European Systems


Required: Software designed for operation on European systems should be compatible with a 21-pin RGB cable (15kHz RGB output mode).

Enquire about technical problems here (edts@soe.sega.co.uk).

In France, there are many televisions that use 15kHz RGB scart cables, but not many televisions with PAL video input jacks.

On the other hand, Dreamcast picture output is by video/15kHzRGB/VGA exclusive output.

For this reason, unless 15kHz RGB is consciously supported by the software side, software cannot be played in most general households in France.

 

Recommended: The option for to switch to a 60Hz screen on European software.

If the software provides the user with this option, the following rules are REQUIRED
    1. 50Hz (PAL) is the Default setting.
    2. A facility to test that the user’s TV is capable of a 60Hz refresh rate must be included. This test procedure must involve setting the screen to 60Hz for no less than 5 seconds and then returning to 50Hz mode and asking the user if the image displayed correctly.

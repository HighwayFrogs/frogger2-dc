
Tool to display library version of SHINOBI (sbver.exe)

This tool picks out the character strings of Shinobi Library version
number, which had been used, from application binary file created by
using Shinobi Library, and displays them.
By inputting the command below in command line of MS-DOS

	sbver <filename>

version number of Shinobi Library which had been used is displayed
as shown below.

(Example)
c:\>sbver 1st_read.bin
